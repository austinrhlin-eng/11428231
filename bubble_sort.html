Bubble Sort - 氣泡排序法

操作原理

氣泡排序透過反覆比較相鄰元素，將較大的元素交換到右側，像氣泡一樣逐步浮到序列末端。每一輪迴圈會把當前未排序區間中的最大值移到尾端，直到整個陣列排序完成。

時間和空間複雜度

情況 複雜度 說明
最佳情況 O(n) 當陣列已經排序或接近排序時，只需一次掃描即可完成。
平均情況 O(n2) 一般無序陣列需要雙層迴圈比較與交換。
最壞情況 O(n2) 完全逆序陣列需要最多交換與比較次數。
空間複雜度 O(1) 僅使用常數級額外空間來暫存交換值。
穩定程度 穩定 相等元素不會改變相對順序。

演算法偽代碼與互動視覺化

演算法偽代碼

function bubbleSort(arr):
n = arr.length
for i = 0 to n - 2:
swapped = false
for j = 0 to n - 2 - i:
if arr[j] > arr[j + 1]:
swap(arr[j], arr[j + 1])
swapped = true
if not swapped: break
