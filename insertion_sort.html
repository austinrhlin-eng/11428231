Insertion Sort - 插入排序法

操作原理

插入排序法將陣列分成已排序和未排序兩部分，每次從未排序部分取出第一個元素，插入到已排序部分的適當位置。

時間和空間複雜度

情況 複雜度 說明
最佳情況 O(n) 當陣列已經排序時，只需一次掃描即可完成。
平均情況 O(n2) 一般無序陣列需要多次比較與移動。
最壞情況 O(n2) 完全逆序陣列需要最多比較與移動次數。
空間複雜度 O(1) 僅使用常數級額外空間來暫存交換值。
穩定程度 穩定 相等元素不會改變相對順序。

演算法偽代碼與互動視覺化

演算法偽代碼

function insertionSort(arr):
for i = 1 to n - 1:
current = arr[i]
j = i - 1
while j >= 0 and arr[j] > current:
arr[j + 1] = arr[j]
j = j - 1
arr[j + 1] = current
