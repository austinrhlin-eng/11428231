Selection Sort - 選擇排序法

操作原理

選擇排序每一輪都在未排序區間中找出最小值，然後與未排序區間的第一個元素交換。如此反覆進行，左側區間會逐步成為已排序狀態，直到整個陣列排序完成。

時間和空間複雜度

情況 複雜度 說明
最佳情況 O(n2) 即使陣列已排序，仍需完整掃描尋找最小值。
平均情況 O(n2) 每輪都要在剩餘元素中找最小值。
最壞情況 O(n2) 比較次數固定為二次方等級。
空間複雜度 O(1) 只需要常數額外空間記錄索引與交換暫存。
穩定程度 不穩定 交換動作可能改變相等元素的相對順序。

演算法偽代碼與互動視覺化

演算法偽代碼

function selectionSort(arr):
for i = 0 to n - 2:
minIndex = i
for j = i + 1 to n - 1:
if arr[j] < arr[minIndex]:
minIndex = j
if minIndex != i: swap(arr[i], arr[minIndex])
return arr
