Merge Sort - 合併排序法

操作原理

合併排序法採用分治法，先把陣列持續切成左右兩半，直到每個區段只剩一個元素，再把已排序的小區段逐步合併成完整有序陣列。它的核心是在合併時比較左右兩邊目前最前面的元素，依序取出較小者放回原陣列。

時間和空間複雜度

情況 複雜度 說明
最佳情況 O(n log n) 不論資料是否接近排序，分割與合併流程都會進行。
平均情況 O(n log n) 每一層遞迴都需要線性合併，總共有 log n 層。
最壞情況 O(n log n) 即使完全逆序，複雜度仍維持在 n log n。
空間複雜度 O(n) 需要額外暫存陣列來完成合併。
穩定程度 穩定 相等元素在合併時可維持原有順序。

演算法偽代碼與互動視覺化

演算法偽代碼

function mergeSort(arr) {
if (arr.length <= 1) return arr;
const mid = Math.floor(arr.length / 2);
const left = mergeSort(arr.slice(0, mid));
const right = mergeSort(arr.slice(mid));
return merge(left, right);
}
function merge(left, right) {
const result = [];
while (left.length && right.length) result.push(left[0] <= right[0] ? left.shift() : right.shift());
return result.concat(left, right);
}
